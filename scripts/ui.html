⚠️


    

      
A保持代理运行：每次用 Claude Code 前，先双击 start.bat 或 node proxy.js（本界面能打开就说明它开着）。

      
B打开目标文件：复制上方「目标文件」路径，在资源管理器地址栏粘贴并回车；右键 settings.json → 用记事本打开（没有这个文件就新建一个同名空白文件）。

      
C粘贴保存：把刚复制的内容粘进去。新手（文件是空的/不存在）直接整段保存；老手（已有配置）只贴 env 那段、保留其它内容。

      
D完全退出并重启 Claude Code：关闭 VS Code 里的 Claude Code 面板 / 退出终端再重开，新配置才会生效（热重载不一定生效）。

      
E验证是否成功：在 Claude Code 发任意一条消息，回到本界面看「活动日志」是否出现记录；或点「🔧 运行自检」。没记录 = 配置没生效，重做 B~D。
